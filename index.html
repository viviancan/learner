Welcome to Principles of User Centered Web Design

Course Description

In this course, you will learn user centered web design principles. Surprisingly, these concepts are similar to those found in learner centered design. These principles can be easily integrated to your work flow to help you create beautiful, intuitive, and functional web sites.

The course is introductory level, and should not take more than 30 minutes. The course will be tailored based on your preferred learning style, which will be determined by your results from the VARK questionnaire.

Course Outline

  1. VARK Questionnaire
  2. What is User Centered Design
  3. Foundations of User Centered Design
  4. The User Centered Design Process
  5. Conclusion
